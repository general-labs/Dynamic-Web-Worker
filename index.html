Web Worker

Web Workers are built in browser technology. No library required to use it. You can use it for any web application.





Things you can do with Web Worker

Font caching

Network polling and web sockets

Data and web page pre-caching, caching

Local database (indexedDB) operations

Virtual DOM diffing

Computationally intensive data operations

Gaming. Canvas drawing and image processing

Much more...
